## Image Guidelines:

- **Format**: PNG with transparent background (preferred) or SVG
- **Size**: Maximum 400x400px recommended
- **Aspect Ratio**: Logos will be scaled to fit while maintaining aspect ratio
- **File Size**: Keep under 100KB per image for optimal performance
- **Background**: Transparent background works best

## Note:
If a logo image is not found, the partner name will be displayed as text fallback.
